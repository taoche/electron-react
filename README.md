# electron-react
a electron boilerplate and use react

### Feature list
React 16

Webpack 4

Typescript

Electron 2.x

Hot reload

Electron-builder

Multi Platform Build

Better two package.json Structure

MacOS Linux(Manjaro) Windows(7, 10)



#### Getting Started

```

# Install dependencies
yarn

# Development
yarn dev

# Build
yarn build (current platform)

```
